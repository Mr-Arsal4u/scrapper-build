# Quick Start - No Virtual Environment

## ✅ Simple Commands (No venv needed!)

### Run Web App:
```bash
python3 app.py
```
Or:
```bash
./run.sh
```

### Run Simple Scraper:
```bash
python3 simple_scraper.py
```

### Setup VPN Extension:
```bash
./setup_vpn_extension.sh
```

---

## 📋 Complete Workflow

1. **Install dependencies** (one time):
   ```bash
   pip3 install -r requirements.txt
   ```

2. **Run the app**:
   ```bash
   python3 app.py
   ```

3. **Install VPN extension** (when Chrome opens):
   - Go to Chrome Web Store
   - Install VPN extension (Windscribe, TunnelBear, etc.)
   - Connect VPN
   - Close Chrome

4. **Run again** - VPN works automatically!
