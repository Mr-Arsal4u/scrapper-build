🔧 VPN Extension Setup

One-time setup to enable scraping

📝 Note: This is a one-time setup. After completing these steps, the scraper will run continuously and automatically scrape every 5 minutes.
1

Click "Setup VPN Extension" Button

This will open Chrome with the scraper's profile. Chrome Web Store will open automatically.

2

Install VPN Extension

In the Chrome window that opens, search for "VPN" and install a free VPN extension:

  • Windscribe Free (recommended - 10GB/month)
  • TunnelBear Free (500MB/month)
  • ProtonVPN (if available)
  • Any free VPN extension
3

Connect VPN

Click the VPN extension icon in Chrome toolbar → Click "Connect" or "Turn On" → Wait for connection (icon should show "Connected")

4

Verify VPN is Working

Visit https://api.ipify.org in Chrome - it should show a different IP address. Then close Chrome.

5

Click "Ready to Scrape" Button

After completing the VPN setup, click the "Ready to Scrape" button below. The scraper will start running continuously!
